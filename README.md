# 中国音乐流媒体热门歌曲排行榜

整合中国各大音乐流媒体平台的热门歌曲数据，按播放量排序展示前100首热歌。

**在线访问**: https://madeye.github.io/hot-music-china/

## 数据来源

- [QQ音乐热歌榜](https://y.qq.com/n/ryqq/toplist/26)
- [网易云音乐飙升榜](https://music.163.com/discover/toplist)
- [酷狗音乐TOP500](https://www.kugou.com/yy/html/rank.html)
- [腾讯音乐由你榜](https://www.tencentmusic.com/zh-cn/uni-chart.html)
- [Apple Music中国榜](https://music.apple.com/cn/playlist/每周热门-100-首-中国大陆/pl.939cf56e73c44970b81fd9648f859223)

## 功能特性

- 展示100首热门歌曲，按综合热度排序
- 支持按平台筛选（QQ音乐、网易云、酷狗、抖音）
- 支持歌曲名和歌手搜索
- 每首歌曲提供多平台播放链接，点击直达
- 响应式设计，支持移动端访问

## 技术栈

- 纯静态HTML/CSS/JavaScript
- 无需后端服务
- GitHub Pages托管

## 声明

本项目数据整合自各平台公开榜单，仅供学习参考。播放量为综合热度指数，非精确播放次数。
